# avr-baremetal
Learning AVR microcontrollers

# Hardware
- AVR2560
- USBASP
- LSM60S0

# Goals
- [x] LED blinky
- [] Switch input and LED blinky
- [] External interrupt using switch
- [] UART: polling and interrupt
- [] ADC
- [] Timer counter
- [] Timer interrupt
- [] I2C with LSM60S0: polling and interrupt
- [] SPI with LSM60S0: polling and interrupt
- [] Watchdog timer
- [] Flash memory
